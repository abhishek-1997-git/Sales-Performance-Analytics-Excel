# Sales Performance Analytics Dashboard

## Overview
This project is an interactive Sales Performance Analytics Dashboard built in Microsoft Excel using 500 rows of sales data. The dashboard provides insights into sales trends, customer behavior, product performance, and business KPIs.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas

## Key Features
- Sales Analysis
- Customer Insights
- Product Performance Tracking
- KPI Dashboard
- Interactive Filters

## Skills Demonstrated
- Data Cleaning

- Dashboard Design
- Business Reporting
- Data Visualization
